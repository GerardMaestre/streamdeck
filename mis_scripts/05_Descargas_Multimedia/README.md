# Descargas Multimedia

Herramientas avanzadas para bajar contenido de internet, como videos o música.

### Scripts Incluidos:
- **`Descargador_Maestro.py`**: Un script súper completo que permite descargar audio (MP3 a máxima calidad 320kbps) y video de la mayoría de las plataformas web.
- **`Servidor_Descargar.py`**: Configuración de red para convertir tu PC en un servidor de hospedaje de descargas temporales.
- Archivos `.exe` (`ffmpeg.exe`, `yt-dlp.exe`): Son los motores internos bajo el capó que hacen posible que el `Descargador_Maestro` baje a máxima velocidad y convierta los formatos.
