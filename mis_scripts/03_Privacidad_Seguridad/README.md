# Privacidad y Seguridad

Scripts para proteger tu identidad, tus datos y administrar el acceso a tu computadora.

### Scripts Incluidos:
- **`Cifrador_De_Carpetas.py` / `descifrador.py`**: Herramientas para proteger tus carpetas con contraseña y cifrado pesado, y descifrarlas cuando las necesites.
- **`Cazador_Intrusos.py`**: Monitorea tu red en busca de dispositivos no autorizados conectados a tu Wi-Fi.
- **`Revisor_Puertos_Abiertos.py`**: Escanea tu computadora para asegurar que no haya puertos vulnerables abiertos a internet.
- **`Identidad_Falsa.py` / `MAC_Spoofer.py`**: Cambian la dirección MAC real de tu tarjeta de red para anonimato.
- **`Destructor_metadatos.py`**: Elimina metadatos (ubicación, cámara, autor) de tus fotos y documentos.
- **`Panic_Button.py`**: Utilidad de emergencia para cerrar aplicaciones, cifrar archivos o bloquear la PC rápidamente.
- **`Adblock.py`**: Bloqueador de anuncios mediante modificaciones seguras a nivel de sistema (`hosts`).
- **`Asesino_Zombies.bat`**: Finaliza procesos invisibles o "zombies" que se quedan colgados en segundo plano.
