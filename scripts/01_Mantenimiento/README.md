# Mantenimiento de Windows

Esta carpeta contiene scripts orientados a limpiar, actualizar y preparar tu sistema operativo.

### Scripts Incluidos:
- **`Activar_Windows_11_Universal.bat`**: Script de activación para Windows 11.
- **`Actualizar_Aplicaciones.bat`**: Se encarga de buscar y descargar las últimas actualizaciones de tus aplicaciones.
- **`Desinstalador_Telemetria.bat`**: Elimina componentes de rastreo de Microsoft para proteger tu privacidad.
- **`Desinstalador_Bloatware.bat`**: Remueve aplicaciones preinstaladas innecesarias (bloatware) que consumen recursos.
- **`Salud_Disco.bat`**: Ejecuta diagnósticos para comprobar y reparar errores en el disco duro.
- **`PC.bat`**: Herramientas generales de mantenimiento del sistema.
- **`god_mode.bat`**: Habilita el "Modo Dios" de Windows, creando un acceso directo a todas las configuraciones avanzadas.
